# Tips Inbox

Append-only capture of coding tips from videos, articles, DMs, etc.
Each entry is a URL + optional note. Process with `/process-tips`.

**Entry format** (anything under the `---` marker is an entry):

```
## 2026-04-09T14:32 — optional short title
Source: https://www.tiktok.com/@someone/video/1234567890
Note: Claude Code hook trick — run a formatter on every file write via PostToolUse
Tags: #claude-code #hooks
```

**Minimum viable entry** (when you're in a hurry):

```
## 2026-04-09
https://www.tiktok.com/@someone/video/1234567890
```

Bare URLs get transcribed automatically during `/process-tips`. A 1-sentence note from you makes processing 10× faster and more accurate, so prefer that when you have the 5 seconds to type it.
